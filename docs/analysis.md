# 分析模块详解

本文档详细介绍 Audio Lab 各分析模块的算法原理、输入输出和实现细节。

---

## 一、特征提取模块

### 1.1 波形提取

**功能：** 提取音频时域波形

**输入：**
- 音频信号 $y$：numpy array
- 采样率 $sr$：int

**输出：**
- 时间轴 $t$：numpy array
- 波形值 $waveform$：numpy array

**算法：**
```python
t = np.arange(len(y)) / sr
waveform = y
```

### 1.2 FFT 频谱分析

**功能：** 计算频域幅度谱

**输入：**
- 音频信号 $y$
- 采样率 $sr$

**输出：**
- 频率轴 $freqs$：0 ~ sr/2 Hz
- 幅度谱 $mag$：各频率分量的幅度

**算法：**
$$X[k] = \sum_{n=0}^{N-1} x[n] e^{-j \frac{2\pi}{N} kn}$$

$$mag[k] = |X[k]| / N$$

### 1.3 STFT 语谱图

**功能：** 计算短时傅里叶变换语谱图

**参数：**
- `n_fft`：FFT窗口大小，默认 2048
- `hop_length`：跳跃长度，默认 512

**输出：**
- 频率轴 $f$
- 时间轴 $t$
- 语谱图 $|Z_{xx}|$

**时间-频率分辨率权衡：**
- 大窗口：高频率分辨率，低时间分辨率
- 小窗口：低频率分辨率，高时间分辨率

### 1.4 Mel 语谱图

**功能：** 计算 Mel 刻度语谱图

**参数：**
- `n_mels`：Mel 滤波器数量，默认 128

**流程：**
1. 计算 STFT 功率谱
2. 应用 Mel 三角滤波器组
3. 取对数转换为 dB

**Mel 刻度转换：**
$$m = 2595 \log_{10}(1 + f/700)$$

### 1.5 MFCC 提取

**功能：** 提取 Mel 倒谱系数

**参数：**
- `n_mfcc`：MFCC 数量，默认 20

**流程：**
1. 计算 Mel 语谱图
2. 应用 DCT
3. 保留低阶系数

## 二、时间序列分析模块

### 2.1 ACF 计算

**功能：** 计算自相关函数

**输入：**
- 序列 $x$
- 滞后阶数 $nlags$

**输出：**
- 滞后阶 $lags$
- ACF 值 $acf\_vals$
- 95% 置信区间 $ci = 1.96/\sqrt{n}$

**公式：**
$$\rho_k = \frac{\sum_{t=k+1}^{n} (x_t - \bar{x})(x_{t-k} - \bar{x})}{\sum_{t=1}^{n} (x_t - \bar{x})^2}$$

### 2.2 PACF 计算

**功能：** 计算偏自相关函数

**算法：** Yule-Walker 方程求解

**应用：** ARIMA 模型定阶

### 2.3 周期性分析

**功能：** 检测信号的周期性

**输出：**
- 主频率 $dominant\_frequency$
- 主周期 $dominant\_period$
- ACF 峰值位置

**方法：**
1. FFT 找主频率
2. ACF 找峰值
3. 综合判断周期

### 2.4 复杂度分析

**功能：** 计算信号复杂度指标

**输出：**
- 零交叉率 $zero\_crossing\_rate$
- 样本熵 $sample\_entropy$
- 是否有调 $is\_tonal$

**样本熵计算：**
$$SampEn = -\ln(A/B)$$

### 2.5 白噪声检验

**功能：** 综合检验信号是否为白噪声

**包含检验：**
1. Ljung-Box 检验
2. Box-Pierce 检验
3. Jarque-Bera 正态性检验
4. ACF 置信区间检验
5. 方差平稳性检验
6. 游程检验

**判定规则：**
- 多数投票：至少 3/5 检验通过
- 输出综合评估

## 三、动态分析模块

### 3.1 动态特征提取

**功能：** 提取音频的时变特征

**输出特征：**

| 特征 | 说明 | 计算方法 |
|------|------|----------|
| 能量 | 短时能量 | $\sum x^2$ |
| 亮度 | 频谱质心 | $\sum f \cdot |X| / \sum |X|$ |
| 复杂度 | 频谱扩散度 | $\sqrt{\sum (f-c)^2 |X| / \sum |X|}$ |
| 节奏 | 节拍强度 | 自相关峰值 |

**参数：**
- `window_size`：分析窗口，默认 0.5 秒
- `hop_size`：跳跃长度，默认 0.25 秒

### 3.2 结构段检测

**功能：** 检测音频的结构变化点

**方法：**
1. 计算相邻窗口的特征差异
2. 差异超过阈值标记为边界
3. 合并相邻小段

**应用：**
- 音乐结构分析（主歌/副歌）
- 语音分段

## 四、波动率分析模块

### 4.1 滚动波动率

**功能：** 计算时变波动率

**公式：**
$$\sigma_t = \sqrt{\frac{1}{w} \sum_{i=t-w+1}^{t} (x_i - \bar{x})^2}$$

**参数：**
- `rolling_window`：滚动窗口大小，默认 10

### 4.2 GARCH(1,1) 建模

**功能：** 对波动率序列建模

**模型：**
$$\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2$$

**输出：**
- 模型参数 $\omega, \alpha, \beta$
- 条件波动率序列
- 持久性指标 $\alpha + \beta$

**拟合方法：** 最大似然估计

## 五、预测模块

### 5.1 ARIMA 预测

**流程：**
```
1. 检验平稳性（ADF检验）
2. 自动定阶（AIC最小）
3. 拟合模型
4. 滚动预测
```

**输出：**
- 预测序列
- 误差指标（RMSE, MAE, MSE）

### 5.2 HMM 预测

**流程：**
```
1. 初始化HMM参数
2. Baum-Welch训练
3. 维特比解码
4. 基于状态预测
```

**参数：**
- `n_states`：隐状态数，默认 3

### 5.3 LSTM 预测

**流程：**
```
1. 构建滑动窗口数据集
2. 划分训练/验证集
3. 训练LSTM网络
4. 早停防止过拟合
5. 滚动预测
```

**网络结构：**
- 输入层：序列长度 × 特征维度
- LSTM层：64 隐藏单元 × 2 层
- 输出层：全连接层

### 5.4 Transformer 预测

**流程：**
```
1. 添加位置编码
2. 多头自注意力
3. 前馈网络
4. 输出预测
```

**网络结构：**
- d_model = 64
- n_heads = 4
- n_layers = 2

## 六、频带分析模块

### 6.1 频带划分

**功能：** 将 Mel 语谱图按频率划分为多个频带

**默认划分：**

| 频带 | Mel 范围 | 大致频率范围 |
|------|----------|--------------|
| 低频带 | 0-42 | 0-500 Hz |
| 中低频带 | 43-85 | 500-2000 Hz |
| 中高频带 | 86-106 | 2000-5000 Hz |
| 高频带 | 107-127 | 5000+ Hz |

### 6.2 分频带预测

**流程：**
```
For each 频带:
    1. 提取该频带的时间序列
    2. 对每个模型进行预测
    3. 计算误差指标
    4. 记录最佳模型
```

### 6.3 可预测性排名

**功能：** 按可预测性对频带排序

**指标：** 平均 RMSE

**输出：**
- 排名列表
- 各频带最佳模型
- 各频带误差统计

## 八、双音频对比模块

### 8.1 DTW 距离

**功能：** 计算两个音频的 DTW 距离

**算法：** 动态规划

$$D(i,j) = d(x_i, y_j) + \min\{D(i-1,j), D(i,j-1), D(i-1,j-1)\}$$

### 8.2 动态相似度

**功能：** 比较两个音频的动态特征

**指标：**
- 能量相似度
- 亮度相似度
- 复杂度相似度
- 节奏相似度

### 8.3 波动率相似度

**功能：** 比较两个音频的波动特性

**方法：** GARCH 参数距离

**内容：** FFT 幅度谱

### 9.3 语谱图

**内容：** STFT 或 Mel 语谱图

**内容：** 真实值与预测值对比

**包含：**
- 真实序列
- 各模型预测
- 图例

### 9.5 误差柱状图

**内容：** 各模型误差对比

**指标：**
- RMSE
