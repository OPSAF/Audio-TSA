# 结果解读

本文档帮助用户理解 Audio Lab 生成的各类图表和统计指标。

---

## 一、概览统计

### 1.1 基本信息

| 指标 | 说明 | 正常范围 |
|------|------|----------|
| 音频时长 | 音频总长度 | 依输入而定 |
| 采样率 | 每秒采样点数 | 8000-48000 Hz |
| 预测步长 | 预测未来多少步 | 10-50 步 |

## 二、时间序列分析结果

### 2.1 周期性分析

**主频率（Dominant Frequency）**
- 含义：信号中最强的频率分量
- 解读：
  - 音乐：对应主音调/基频
  - 语音：对应基音频率（pitch）
  - 噪声：无明显主频率

**主周期（Dominant Period）**
- 含义：主频率对应的周期
- 计算：$T = 1/f$
- 解读：信号重复的时间间隔

**ACF 峰值**
- 含义：自相关函数的峰值位置
- 解读：峰值位置对应周期长度

### 2.2 复杂度分析

**零交叉率（Zero-Crossing Rate）**
- 含义：信号穿过零点的频率
- 解读：
  - 高值：高频成分多，如噪声
  - 低值：低频成分主导，如低音

**样本熵（Sample Entropy）**
- 含义：时间序列的规则性度量
- 解读：
  - < 0.5：规则、可预测（有调信号）
  - 0.5-1.0：中等复杂度
  - > 1.0：复杂、随机（噪声）

**是否有调（Is Tonal）**
- 判断标准：样本熵 < 0.5
- 解读：有调信号有明显音高，噪声无音高

### 2.3 谱平坦度

**含义：** 衡量频谱的"平坦程度"

**解读：**
- 接近 0：有明显的音调成分（谐波结构）
- 接近 1：类似白噪声（平坦频谱）

**应用：**
- 音乐/语音检测
- 音调强度评估

## 三、白噪声检验结果

### 3.1 检验方法说明

| 检验 | 原假设 H₀ | 判断标准 |
|------|-----------|----------|
| Ljung-Box | 自相关全为零 | p > 0.05 通过 |
| Box-Pierce | 自相关全为零 | p > 0.05 通过 |
| Jarque-Bera | 数据服从正态分布 | p > 0.05 通过 |
| ACF Test | ACF值在置信区间内 | 比例 < 10% 通过 |
| 方差平稳性 | 方差恒定 | 比值 < 10 通过 |
| 游程检验 | 序列随机 | p > 0.05 通过 |

### 3.2 综合评估

**判定规则：**
- 多数投票：至少 3/5 检验通过
- 结论：数据是否为白噪声

**解读：**
- **是白噪声**：
  - 预测难度大
  - 建议关注频带分析找可预测成分
- **非白噪声**：
  - 存在可利用的模式
  - 预测模型可能有效

### 3.3 p 值解读

- p > 0.05：不能拒绝原假设（通过检验）
- p < 0.05：拒绝原假设（未通过检验）
- p 值越小，拒绝原假设的证据越强

## 四、预测结果解读

### 4.1 误差指标

**RMSE（均方根误差）**
$$RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$$

- 单位与原数据相同
- 对大误差敏感
- 越小越好

**MAE（平均绝对误差）**
$$MAE = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$$

- 单位与原数据相同
- 对异常值不敏感
- 越小越好

**MSE（均方误差）**
$$MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

- 单位是原数据单位的平方
- 用于优化目标
- 越小越好

### 4.2 模型对比

**选择建议：**
1. 综合考虑三个误差指标
2. RMSE 通常最重要（惩罚大误差）
3. 若模型间差异小，选择简单模型

**常见情况：**
- ARIMA 最好：信号线性、平稳
- LSTM 最好：信号复杂、长程依赖
- Transformer 最好：全局依赖强
- HMM 最好：状态切换明显

## 五、频带分析结果

### 5.1 频带划分

| 频带 | 频率范围 | 典型内容 |
|------|----------|----------|
| 低频带 | 0-500 Hz | 基频、低音 |
| 中低频带 | 500-2000 Hz | 元音、主旋律 |
| 中高频带 | 2000-5000 Hz | 辅音、泛音 |
| 高频带 | 5000+ Hz | 高频噪声、细节 |

### 5.2 可预测性排名

**解读：**
- 排名靠前：该频带规律性强，易于预测
- 排名靠后：该频带随机性强，难以预测

**应用：**
- 选择最佳建模频带
- 理解信号结构
- 优化预测策略

### 5.3 各频带最佳模型

**解读：**
- 不同频带可能适合不同模型
- 低频通常更可预测
- 高频通常更随机

## 六、动态分析结果

### 6.1 能量包络

**含义：** 信号能量随时间的变化

**解读：**
- 高能量：响度大
- 低能量：响度小或静音
- 变化剧烈：动态范围大

### 6.2 频谱质心（亮度）

**含义：** 频谱的"重心"位置

**解读：**
- 高质心：声音"明亮"
- 低质心：声音"暗淡"

### 6.3 频谱扩散度（复杂度）

**含义：** 频谱围绕质心的分散程度

**解读：**
- 高扩散度：频谱分布广，声音丰富
- 低扩散度：频谱集中，声音单调

### 6.4 节拍强度

**含义：** 节奏感的强度

**解读：**
- 高值：节奏明显
- 低值：节奏不明显或无节奏

## 七、波动率分析结果

### 7.1 GARCH 参数

**ω（omega）**
- 基准波动率
- 越大表示基础波动越大

**α（alpha）**
- 对过去冲击的反应
- 越大表示对近期波动越敏感

**β（beta）**
- 波动率的持续性
- 越大表示波动影响持续时间越长

**持久性（α + β）**
- 接近 1：高持久性，波动聚集
- 远小于 1：低持久性，波动快速衰减

### 7.2 条件波动率序列

**含义：** 时变的波动率估计

**解读：**
- 高波动期：信号变化剧烈
- 低波动期：信号相对稳定

## 八、图表解读

### 8.1 波形图

**观察要点：**
- 振幅范围
- 周期性
- 对称性
- 异常值

### 8.2 FFT 频谱图

**观察要点：**
- 主频率位置
- 谐波结构
- 频谱包络
- 低频/高频能量分布

### 8.3 语谱图

**观察要点：**
- 时间-频率模式
- 持续音（水平线）
- 瞬态（垂直线）
- 谐波轨迹

### 8.4 Mel 语谱图

**观察要点：**
- Mel 刻度下的频率分布
- 能量集中区域
- 时间变化模式

### 8.5 ACF/PACF 图

**观察要点：**
- ACF 截尾/拖尾
- PACF 截尾/拖尾
- 置信区间外的显著滞后

**定阶规则：**
- ACF q 阶截尾 → MA(q)
- PACF p 阶截尾 → AR(p)
- 都拖尾 → ARMA

### 8.6 预测对比图

**观察要点：**
- 预测曲线与真实曲线的贴合度
- 各模型的差异
- 预测偏差的方向

## 九、常见问题诊断

### 9.1 预测效果差

**可能原因：**
1. 信号接近白噪声
2. 数据量不足
3. 模型选择不当
4. 超参数需要调优

**解决方法：**
1. 检查白噪声检验结果
2. 增加数据量或使用数据增强
3. 尝试其他模型
4. 进行超参数优化

### 9.2 模型过拟合

**表现：**
- 训练误差小，测试误差大
- 预测曲线过于复杂

**解决方法：**
1. 增加正则化
2. 减小模型复杂度
3. 增加训练数据
4. 使用早停

### 9.3 结果不稳定

**可能原因：**
1. 随机种子未固定
2. 数据划分不同
3. 初始化不同

**解决方法：**
1. 固定随机种子
2. 多次运行取平均
3. 使用交叉验证
