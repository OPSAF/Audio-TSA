## 一、Blues 风格特征

### 1.1 音乐学特征

蓝调（Blues）起源于 19 世纪末的美国南部，是摇滚、爵士、R&B 等多种现代音乐风格的源头：

| 特征 | 描述 |
|------|------|
| **结构** | 12 小节蓝调 (12-bar blues) |
| **和声** | I-IV-V 和弦进行 |
| **音阶** | 蓝调音阶（降三音、降五音、降七音） |
| **情感** | 忧郁、深沉、叙事性 |
| **典型配器** | 吉他（电/木）、口琴、钢琴、贝斯、鼓 |

### 1.2 预期声学特征

基于 Blues 的音乐学特征，预期 Audio Lab 分析将呈现：

| 维度 | 预期特征 | 理由 |
|------|----------|------|
| **能量** | 中等 (0.10-0.18) | 电声乐器但非重金属 |
| **亮度** | 中等偏低 (1200-1700 Hz) | 中低频为主，吉他和人声突出 |
| **复杂度** | 中等 (10.0-10.5) | 和声进行规整但不复杂 |
| **节奏** | 中等 (0.002-0.008) | Shuffle 节奏 |
| **可预测性** | ★★★★☆ 较高 | 12 小节循环结构高度规整 |

### 1.3 与相似风格的区分

| 特征 | Blues | Jazz | Rock |
|------|-------|------|------|
| 和声复杂度 | 简单 (I-IV-V) | 复杂 (扩展和弦) | 中等 |
| 即兴程度 | 中等 | 高 | 中高 |
| 节奏 | Shuffle | Swing | Straight/Shuffle |
| 情感色调 | 忧郁 | 多变 | 激昂 |
| 结构 | 12-bar 循环 | 32-bar AABA | Verse-Chorus |
